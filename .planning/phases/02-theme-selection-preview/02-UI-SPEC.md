## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Swatch label margin-top, inline icon padding |
| sm | 8px | Swatch label margin, hashtag pill padding, dot indicator gap |
| md | 16px | Default element spacing, swatch row gap, dot nav bottom margin |
| lg | 24px | Section gaps, results header bottom margin |
| xl | 32px | Slide canvas internal padding (top/bottom) |
| 2xl | 36px | Slide canvas internal padding (left/right) |
| 3xl | 48px | Corner SVG decoration size |

Exceptions: 36px used for slide canvas horizontal padding (matching reference implementation exactly); 14px for slide body internal gap (matching reference). These are interior-to-slide-canvas values at the 1080px resolution, not app-level layout tokens. -18px for navigation arrow inset: half of 36px arrow button diameter, derived positioning value for centering arrow on carousel viewport edge.

## Component Inventory

### Theme Picker Row

| Property | Value |
|----------|-------|
| Container | `div.theme-picker-row`, horizontal flex, `overflow-x: auto`, `-webkit-overflow-scrolling: touch` |
| Row gap | 16px between swatches |
| Row padding | 0 horizontal (flush with carousel viewport edges), 16px bottom margin |
| Swatch element | `button.theme-swatch`, 48px wide x 32px tall, border-radius 6px, overflow hidden |
| Swatch interior | Two equal halves side-by-side: left = theme background color, right = theme accent color |
| Swatch border (default) | 1.5px solid #D0CCC4 |
| Swatch border (selected) | 2.5px solid #C8A96E (gold ring) |
| Swatch label position | Centered below swatch, 4px margin-top |

## Page Flow (post-upload)

Vertical sequence within the `#results` container, top to bottom:

1. **Results header row** -- filename pill (left) + "Upload new file" link (right) -- 24px bottom margin
2. **Theme picker row** -- horizontal scrollable swatch strip -- 16px bottom margin
3. **Carousel viewport** -- 420x420px centered preview with arrow overlays -- 0px bottom margin
4. **Navigation bar** -- dots (left) + slide counter (right) in a flex row -- 24px bottom margin
5. **Caption section** -- "Caption" heading + caption-block -- 16px bottom margin
6. **Hashtags section** -- "Hashtags" heading + hashtag pills

Items 1, 5, and 6 reuse existing Phase 1 CSS classes: `.results-header`, `.filename-pill`, `.upload-new-link`, `.caption-block`, `.hashtags-container`, `.hashtag-pill`.

Items 2, 3, and 4 are new Phase 2 additions.

The Phase 1 slide cards (`.slide-card` elements), metadata row, and "Slides (N)" section heading are removed/replaced by the carousel.
