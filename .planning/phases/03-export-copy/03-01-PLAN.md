  1. Add click handler for the SINGLE SLIDE download button (`singleBtn`). Inside renderResults, after the export bar is appended:
     - On click: call `setExportLoading(singleBtn, true)`, then render the current slide.
     - Rendering approach: Get the current `.slide-frame` element via `slides[currentSlide]`. The slide-frame is inside `.slide-canvas` which has `transform: scale(0.389)`. To capture at full 1080x1080:
       a. Clone the entire `slideCanvas` element using `slideCanvas.cloneNode(true)`.
       b. On the clone, set `style.transform = 'none'` to remove the scale.
       c. On the clone, set `style.position = 'absolute'`, `style.left = '-9999px'`, `style.top = '0'`, `style.width = '1080px'`, `style.height = '1080px'`.
       d. Hide all `.slide-frame` children in the clone except the one at index `currentSlide` (set `display: none` on others, set `display: block` plus classes `active` on the target).
       e. Append clone to `document.body`.
       f. Call `html2canvas(clone, { width: 1080, height: 1080, scale: 1, useCORS: true, logging: false })`.
       g. In the `.then(function(canvas) {...})` callback: convert canvas to blob via `canvas.toBlob(function(blob) {...}, 'image/png')`, create a download link with `URL.createObjectURL(blob)`, set the download attribute to `makeTopicSlug(parsed.field) + '-slide-' + padIndex(currentSlide) + '.png'` where `padIndex` zero-pads to 2 digits (e.g. 01, 02). Trigger click on the link, revoke the object URL, remove the clone from document.body, call `setExportLoading(singleBtn, false)`.
       h. Add a `.catch(function(err) {...})` that calls `alert('Export failed. Try reloading the page.')`, removes clone, and calls `setExportLoading(singleBtn, false)`.

  2. Add a helper `padIndex(i)` that returns `(i + 1) < 10 ? '0' + (i + 1) : '' + (i + 1)` -- converts 0-based index to 1-based zero-padded string.

  3. Add click handler for the ZIP download button (`zipBtn`). On click:
     - Call `setExportLoading(zipBtn, true)`.
     - Create a new `JSZip()` instance.
     - Create a `var slug = makeTopicSlug(parsed.field)`.
     - Iterate over all slides (0 to totalSlides-1). For each slide index, render it using the same clone approach as single-slide but capturing each slide-frame in sequence.
     - Use a sequential Promise chain (not Promise.all, to avoid DOM contention): start with `var chain = Promise.resolve()` then for each slide index `chain = chain.then(function() { return renderSlideToBlob(i); })` where `renderSlideToBlob(i)` returns a Promise that:
       a. Clones slideCanvas, positions off-screen, shows only slide `i`, calls html2canvas with same options.
       b. Returns a Promise wrapping `canvas.toBlob` that resolves with the blob.
       c. Adds the blob to the zip instance: `zip.file(slug + '-slide-' + padIndex(i) + '.png', blob)`.
       d. Removes the clone from document.body.
     - After the chain completes, call `zip.generateAsync({ type: 'blob' })`, then create a download link for the ZIP blob with filename `slug + '-carousel.zip'`, trigger download, revoke URL, call `setExportLoading(zipBtn, false)`.
     - On error (catch): `alert('Export failed. Try reloading the page.')` and `setExportLoading(zipBtn, false)`.
