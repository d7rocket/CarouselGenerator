**Part 2 - Markdown parser (INPUT-03, INPUT-04, INPUT-05, INPUT-06):**
`function parseMarkdown(text)` returns an object with this exact shape:
```javascript
{
  title: string,        // from "# Title" line (line 1)
  date: string,         // from "**Date:** VALUE" in metadata line
  field: string,        // from "**Field:** VALUE" in metadata line
  sourcesCount: number, // from "**Sources:** N" in metadata line
  slides: [             // from "## Slide N: Title" sections
    { number: number, title: string, body: string }
  ],
  caption: string,      // from "## Caption" section
  hashtags: string[],   // from "## Hashtags" section, split by space, each starts with #
  sources: string[],    // from "## Sources" section, each numbered entry
  images: string[]      // from "## Images" section, each URL
}
```

Parser logic:
1. Split text by lines. Find line starting with `# ` (not `## `) for title: `title = lines.find(l => /^# /.test(l)).replace(/^# /, '').trim()`
2. Find metadata line matching pattern `**Date:**`. Parse with regex: `/\*\*Date:\*\*\s*([^|]+)/` for date, `/\*\*Field:\*\*\s*([^|]+)/` for field, `/\*\*Sources:\*\*\s*(\d+)/` for sourcesCount
3. Split text by `\n## ` to get sections. For each section:
   - If starts with `Slide \d+:` — extract number and title from header, body is remaining text after first line (trimmed). Push to slides array.
   - If starts with `Caption` — body after first line is caption (trimmed)
   - If starts with `Hashtags` — body after first line, split by whitespace, filter to strings starting with `#`
   - If starts with `Sources` — body after first line, split by numbered entries (regex `/^\d+\.\s/m`), filter empty
   - If starts with `Images` — body after first line, extract URLs matching `/https?:\/\/[^\s]+/g`
4. Return the parsed object
